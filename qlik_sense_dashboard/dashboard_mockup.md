## 🎨 **Design Specifications**

### **Color Palette**
```
Primary Colors:
- Primary Blue: #1F77B4 (Headers, titles)
- Secondary Blue: #AEC7E8 (Accents, borders)
- Success Green: #2CA02C (Positive metrics, good performance)
- Warning Orange: #FF7F0E (Caution, needs attention)
- Danger Red: #D62728 (Negative metrics, poor performance)
- Neutral Gray: #7F7F7F (Text, secondary information)

Background Colors:
- Main Background: #FFFFFF (White)
- Card Background: #F8F9FA (Light gray)
- Border Color: #E9ECEF (Light border)
```

### **Typography**
```
Headers:
- Main Title: 24px, Bold, Primary Blue
- Section Headers: 18px, Semi-bold, Dark Gray
- Sub-headers: 16px, Medium, Medium Gray

Body Text:
- Regular Text: 14px, Regular, Dark Gray
- Small Text: 12px, Regular, Medium Gray
- Metric Values: 20px, Bold, Primary Blue
```

### **Layout Guidelines**
```
Spacing:
- Margin: 20px (outer edges)
- Padding: 15px (within cards)
- Gap between elements: 20px
- Card border radius: 8px
- Shadow: Subtle drop shadow for depth

Grid System:
- Use 12-column grid for responsive layout
- Maintain consistent spacing between elements
- Align elements to grid lines for clean appearance
```

## 🔧 **Interactive Elements**

### **Selections & Filters**
```
Global Selections:
- Date Range: Last 7, 14, 30, 90 days
- Platform: Facebook, Google, LinkedIn, TikTok
- Campaign Type: Awareness, Consideration, Conversion
- Geographic Region: North America, Europe, Asia, etc.

Local Selections:
- Device Type: Mobile, Desktop, Tablet
- Ad Format: Video, Image, Carousel, Story
- Campaign Status: Active, Paused, Completed
```

### **Dashboard Actions**
```
Cross-Filtering:
- Click on platform → Filter all charts
- Select date range → Update all views
- Choose campaign → Highlight related data
- Click on country → Filter geographic data

Highlighting:
- Hover over data points → Highlight related information
- Select multiple items → Show relationships
- Click on legend → Isolate specific data
```

## 📱 **Responsive Design**

### **Desktop Layout (1200px+)**
- Full dashboard with all elements visible
- Side-by-side charts and tables
- Detailed information displayed

### **Tablet Layout (768px - 1199px)**
- Stacked layout for better readability
- Simplified charts with key metrics
- Touch-friendly interactive elements

### **Mobile Layout (<768px)**
- Single-column layout
- Simplified charts and tables
- Swipe navigation between dashboard pages
- Touch-optimized controls

## 🚀 **Implementation Notes**

1. **Start with the Executive Summary** - It's the most important dashboard
2. **Use consistent color coding** across all visualizations
3. **Implement progressive disclosure** - Show summary first, details on demand
4. **Test interactivity** thoroughly before finalizing
5. **Optimize for performance** - Use data model optimization for large datasets
6. **Document all expressions** for portfolio presentation
